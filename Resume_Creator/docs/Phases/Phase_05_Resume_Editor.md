# Phase 5: Resume Editor

## Overview
Provide a real-time, WYSIWYG editing experience for the generated resume.

## Key Actions
- Built an **A4-style live editor** using `contentEditable`.
- Integrated **react-to-pdf** for high-quality PDF exports.
- Implemented real-time synchronization between data and view.
- Added formatting controls (Bold, Italic, Lists) and template switching.
